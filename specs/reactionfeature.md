## Feature 3: Reactions Feature

**Purpose:**  
Allow users to express empathy and emotional support quickly without needing to type a reply.

**Expected User:**  
Users who want to show support, encouragement, or understanding in a simple and non-verbal way.

**Main Functionality:**  
Users can react to posts and replies using predefined emotional reactions such as Heart and Care. These reactions are counted and displayed on each post or reply.

**UI Design:**

* Each post and reply includes reaction buttons:
  * ❤️ Heart
  * 🤗 Care
* Reaction icons are displayed below the content
* A counter is shown beside each reaction
* Clicking a reaction highlights the selected icon

**Acceptance Criteria:**

1. Users can react to both posts and replies.
2. The system allows only one reaction per user per post or reply.
3. Users can change or remove their reaction.
4. The system displays the total number of reactions for each type.
5. Reactions must remain anonymous and must not reveal user identity.
